Computation for an N-Body Particle Simulation using MPI for parallelized computations

  For my parallel computing class, I built a program that randomly generates light, medium and heavy particles. The program provides each particle with an initial velocity, a mass and an intial position. This is done in a 3 dimensional plane (x, y, and z plane). The particles interact with each other by evaluating Newton's equations of motion for the forces of all other particles that are acting on each individual particles. Using the matrix of force interactions on each particle and Euler's method, I am able to calculate the next velocity step and the next position step in the sequence. Using command line inputs, the user can choose the amount of all 3 types of particles, the time between steps and the number of steps in the program. After each step, a snapshot is taken of the current position of all particles in 2 dimensions (x and y plane). These pistures are then transferred into a video to show the full particle simulation as shown in the video below. This project was done with Matthew Shortt as a 2 person team. We also added MPI for parallelized computations for greater speedups and efficiency.
